# AIEffect - UI/UX 设计规范

## 1. 设计风格定位

### 1.1 整体风格

**现代极简科技风（AI 蓝紫风）**

- 核心特点：专业、可信、创新、未来感
- 适用场景：AI 工具、大模型产品、云服务
- 视觉感受：护眼、通透、高级感

## 2. 配色系统

### 2.1 核心配色逻辑

| 颜色类型          | 颜色值                | 用途说明                      |
| ----------------- | --------------------- | ----------------------------- |
| **主背景色**      | `#f0f4ff` → `#e8f0ff` | 浅蓝/淡青渐变背景，护眼不刺眼 |
| **主色调 (紫蓝)** | `#6366f1` (Indigo)    | 品牌色、视觉焦点、按钮、图标  |
| **主色调 (渐变)** | `#8b5cf6` → `#6366f1` | 紫蓝渐变，用于强调区域        |
| **辅助色 (青)**   | `#06b6d4` (Cyan)      | 次要强调、成功状态            |
| **强调色 (粉紫)** | `#ec4899` (Pink)      | 高亮、危险、激活状态          |

### 2.2 中性色系统

| 颜色   | Hex       | 用途               |
| ------ | --------- | ------------------ |
| 黑色   | `#0f172a` | 主标题、重要文字   |
| 深灰   | `#334155` | 副标题、正文       |
| 中灰   | `#64748b` | 辅助文字、说明文字 |
| 浅灰   | `#94a3b8` | 占位符、禁用状态   |
| 极浅灰 | `#e2e8f0` | 分割线、边框       |
| 白色   | `#ffffff` | 卡片背景、次要按钮 |

### 2.3 功能色

| 状态 | 颜色 | Hex       |
| ---- | ---- | --------- |
| 成功 | 绿色 | `#10b981` |
| 警告 | 橙色 | `#f59e0b` |
| 错误 | 红色 | `#ef4444` |
| 信息 | 蓝色 | `#3b82f6` |

## 3. 组件设计规范

### 3.1 按钮 (Button)

#### 主按钮 (Primary)

- 背景：紫蓝渐变 `var(--gradient-primary)`
- 文字：白色 `#ffffff`
- 圆角：8px
- 阴影：`var(--shadow-md)`
- Hover：上移 2px + 发光阴影
- Active：下移 1px

```css
.btn-primary {
  background: var(--gradient-primary);
  color: white;
  border: none;
  padding: 10px 20px;
  border-radius: 8px;
  font-weight: 600;
  cursor: pointer;
  transition: all 0.2s ease;
  box-shadow: var(--shadow-md);
}

.btn-primary:hover {
  transform: translateY(-2px);
  box-shadow: var(--shadow-glow), var(--shadow-lg);
}
```

#### 次要按钮 (Secondary)

- 背景：白色 `#ffffff`
- 文字：主色 `var(--primary)`
- 边框：1px `var(--border)`
- 圆角：8px

#### 危险按钮 (Danger)

- 背景：`#ef4444`
- 文字：白色

#### 按钮尺寸

| 尺寸    | 高度 | 内边距    | 字体大小 |
| ------- | ---- | --------- | -------- |
| Small   | 32px | 8px 16px  | 13px     |
| Default | 40px | 10px 20px | 14px     |
| Large   | 48px | 14px 28px | 16px     |

### 3.2 输入框 (Input)

- 背景：白色 `#ffffff`
- 边框：1px `var(--border)`
- 圆角：8px
- 内边距：10px 14px
- Focus：边框变主色 + 发光阴影

```css
.input {
  width: 100%;
  padding: 10px 14px;
  border: 1px solid var(--border);
  border-radius: 8px;
  background: white;
  font-size: 14px;
  transition: all 0.2s ease;
}

.input:focus {
  outline: none;
  border-color: var(--primary);
  box-shadow: 0 0 0 3px rgba(99, 102, 241, 0.1);
}
```

### 3.3 卡片 (Card)

- 背景：白色 `#ffffff`
- 圆角：12px
- 阴影：`var(--shadow-sm)` 或 `var(--shadow)`
- 内边距：20px
- Hover：阴影加深（可选）

```css
.card {
  background: white;
  border-radius: 12px;
  box-shadow: var(--shadow);
  padding: 20px;
  border: 1px solid var(--border-light);
}
```

### 3.4 开关 (Switch)

- 开启状态：主色渐变 `var(--gradient-primary)`
- 关闭状态：浅灰 `#cbd5e1`
- 圆形滑块：白色

### 3.5 标签 (Tag/Badge)

| 类型 | 背景      | 文字      |
| ---- | --------- | --------- |
| 成功 | `#d1fae5` | `#065f46` |
| 警告 | `#fef3c7` | `#92400e` |
| 错误 | `#fee2e2` | `#991b1b` |
| 信息 | `#dbeafe` | `#1e40af` |
| 紫色 | `#ede9fe` | `#5b21b6` |
| 运行中 | `#d1fae5` | `#065f46` |
| 已停止 | `#fee2e2` | `#991b1b` |

## 4. 页面布局规范

### 4.1 整体布局

```
┌─────────────────────────────────────────────────────────┐
│  标题栏 (56px)                                            │
├──────────┬──────────────────────────────────────────────┤
│          │  页面头部 (80px)                              │
│          ├──────────────────────────────────────────────┤
│  侧边栏  │                                               │
│  (260px) │         主内容区域                            │
│          │                                               │
│          ├──────────────────────────────────────────────┤
│          │  底部状态栏 (72px)                            │
└──────────┴──────────────────────────────────────────────┘
```

### 4.2 侧边栏 (Sidebar)

- 宽度：260px
- 背景：白色
- 右侧边框：1px `var(--border)`
- Logo 区域高度：80px
- 导航项高度：48px

**导航项样式**：

- 默认：透明背景，深灰文字
- Hover：浅灰背景 `#f8fafc`
- 激活：紫蓝渐变背景 + 白色文字 + 左侧 4px 高亮条

**导航菜单项**（7个）：
1. 控制面板
2. 字幕视觉
3. 语音配置
4. 记忆管理
5. Agents 工作室
6. 角色管理
7. 运行日志

### 4.3 主内容区域

- 背景：渐变背景 `var(--gradient-bg)`
- 内边距：24px
- 最大宽度：无限制（自适应）

**内容间距**：

- 卡片之间：20px
- 页面标题与内容：24px
- 表单元素之间：16px

### 4.4 栅格系统

采用 12 列栅格，基于 Tailwind CSS：

| 断点 | 宽度     | 列间距 |
| ---- | -------- | ------ |
| sm   | ≥ 640px  | 20px   |
| md   | ≥ 768px  | 20px   |
| lg   | ≥ 1024px | 24px   |
| xl   | ≥ 1280px | 24px   |

## 7. 动效规范

### 7.1 过渡动画

| 属性   | 时长  | 缓动函数 |
| ------ | ----- | -------- |
| 颜色   | 150ms | ease-out |
| 变换   | 200ms | ease-out |
| 阴影   | 200ms | ease-out |
| 透明度 | 150ms | ease-out |

### 7.2 交互反馈

- Hover：轻微上移 + 阴影加深
- Click：按压效果（scale 0.98）
- Loading：旋转加载动画 + 骨架屏

### 7.3 页面过渡

- 淡入淡出：`opacity 0.3s ease`
- 滑动进入：`transform 0.3s ease`

## 8. 各页面设计要点

### 8.1 控制面板

- 状态标签使用圆角胶囊样式
- 配置项分组放入卡片
- 重要按钮（启动服务）使用主按钮
- **服务状态显示**：显示各微服务运行状态（API Gateway、WebSocket、Log Service）
- **端口显示**：显示当前使用的端口（8500-8505）

### 8.2 字幕视觉

- 实时预览区域背景深色
- 预览窗口有悬浮效果
- 滑块控件样式统一
- 打字机效果动画流畅

### 8.3 语音配置

- 标签页使用紫蓝激活状态
- 音量滑块使用主色
- 连接状态使用颜色标识
- 引擎切换动画流畅

### 8.4 记忆管理

- 使用说明区域使用浅黄背景
- 输入框分组清晰
- 操作按钮排列整齐
- 压缩进度可视化

### 8.5 Agents 工作室

#### 8.5.1 基础工作室

- 角色卡片网格布局
- 角色头像圆形展示
- 状态指示器（在线/离线/忙碌）
- 快速操作按钮（编辑、删除、启动）

#### 8.5.2 PixiJS 可视化工作室

- **画布区域**：
  - 背景色：`var(--pixi-bg)` (#1a1a2e)
  - 网格线：`var(--pixi-grid)` (#2d2d44)
  - 网格间距：50px
  - 支持拖拽平移、滚轮缩放

- **节点样式**：
  - 默认节点：`var(--pixi-node-default)` (#6366f1)
  - 激活节点：`var(--pixi-node-active)` (#10b981)
  - 选中节点：`var(--pixi-node-selected)` (#ec4899)
  - 节点圆角：8px
  - 节点阴影：发光效果

- **连接线样式**：
  - 颜色：`var(--pixi-connection)` (#4b5563)
  - 选中时：主色高亮
  - 动画：数据流动效果（虚线动画）

- **交互规范**：
  - 左键拖拽：移动节点/平移画布
  - 滚轮：缩放画布
  - 右键：打开上下文菜单
  - 双击：编辑节点属性
  - 框选：多选节点

### 8.6 角色管理

- 角色列表卡片式布局
- 头像使用圆形裁剪
- 角色属性标签化展示
- 批量操作支持

### 8.7 运行日志

- 日志行使用斑马纹（浅灰交替）
- 日志级别使用颜色标识
- 时间戳使用等宽字体
- 实时滚动跟随
- 支持暂停/继续
