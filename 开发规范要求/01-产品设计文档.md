# Ollama TTS Adapter - 产品设计文档 (PRD)

## 1. 产品概述

### 1.1 产品名称
**AIEffect - AI AI 交互Agent**

### 1.2 产品定位
一款集 Ollama 代理服务、AI 语音合成（TTS）、字幕展示、对话记忆管理于一体的现代化 AI 交互工具，为游戏、视频、桌面应用等提供无缝的 AI 语音集成能力。

### 1.3 核心价值
- 将 Ollama 本地模型转换为 OpenAI 兼容 API
- 提供高质量的 AI 语音合成和字幕展示
- 支持对话历史智能压缩，防止上下文溢出

## 2. 功能模块设计

### 2.1 控制面板（服务连接）

#### 2.1.1 Ollama 代理服务
- **功能描述**：在指定端口提供 Ollama 兼容 API 代理服务
- **核心功能**：
  - 服务启停控制（启动/停止）
  - 端口配置（默认 11434）
  - 服务状态实时显示
  - 外部访问地址展示

#### 2.1.2 API 配置
- **功能描述**：配置 LLM API 接口地址和密钥
- **核心功能**：
  - 服务商预设选择（自定义、OpenAI、Azure 等）
  - API URL 配置
  - API Key 管理（支持显示/隐藏）
  - 模型名称配置
  - 一键复制配置按钮

#### 2.1.3 语音合成
- **功能描述**：语音合成功能开关和基础配置
- **核心功能**：
  - 语音合成总开关
  - 引擎选择（GPT-SoVITS、讯飞语音等）
  - 角色配置（全局默认、男性默认、女性默认）
  - 音量调节
  - 自动播放开关

#### 2.1.4 局域网开放
- **功能描述**：允许局域网内其他设备访问
- **核心功能**：
  - 局域网访问开关
  - 绑定地址说明
  - 修改后重启提示

### 2.2 字幕视觉

#### 2.2.1 窗口控制
- **功能描述**：控制字幕悬浮窗的显示与行为
- **核心功能**：
  - 显示/隐藏字幕
  - 隐藏控件
  - 清空内容
  - 透明/重置窗口

#### 2.2.2 历史记录
- **功能描述**：查看和管理字幕历史
- **核心功能**：
  - 历史记录列表
  - 记录数量统计
  - 历史记录可在字幕窗口中查看

#### 2.2.3 字幕设置
- **功能描述**：自定义字幕样式
- **核心功能**：
  - 字体颜色自定义
  - 背景颜色选择（含预设如科技极光）
  - 不透明度调节（0-100%）
  - 字体大小调节
  - 打字速度调节（毫秒级）

#### 2.2.4 实时预览
- **功能描述**：实时预览字幕效果
- **核心功能**：
  - 字幕页面预览（Galgame 风格悬浮字幕）
  - 流式窗口预览
  - 测试输入与发送功能

### 2.3 语音配置

#### 2.3.1 功能切换
- **功能描述**：在语音合成（TTS）和语音输入（ASR）之间切换
- **核心功能**：
  - TTS/ASR 标签切换

#### 2.3.2 全局控制
- **功能描述**：语音合成全局设置
- **核心功能**：
  - 启用 TTS 输出开关
  - 自动播放开关
  - 保存音频文件开关
  - 播放模式选择（只播放对话等）

#### 2.3.3 混音控制
- **功能描述**：音频混音设置
- **核心功能**：
  - 主音量调节
  - 背景音乐音量（预留）
  - 音效音量（预留）
  - 停止播放/空闲状态显示

#### 2.3.4 引擎管理
- **功能描述**：管理不同的语音合成引擎
- **核心功能**：
  - 引擎标签页（GPT-SoVITS、讯飞语音等）
  - 连接设置（API 地址、测试按钮）
  - 连接状态显示

#### 2.3.5 角色音色管理
- **功能描述**：配置和管理角色音色
- **核心功能**：
  - 当前角色选择
  - 角色名称配置
  - 参考音频目录配置
  - 情感音频配置
  - GPT 模型路径配置

### 2.4 记忆管理

#### 2.4.1 压缩配置
- **功能描述**：配置对话历史的自动压缩策略
- **核心功能**：
  - 存档目录设置（带浏览按钮）
  - 触发阈值设置（消息数量）
  - 压缩数量设置（保留最近 N 条）
  - 检查频率设置（每 N 条检查一次）
  - 压缩前自动备份开关
  - 自动压缩开关
  - 保存配置按钮

#### 2.4.2 角色状态列表
- **功能描述**：查看和管理各角色的对话历史状态
- **核心功能**：
  - 角色列表展示
  - 刷新状态按钮
  - 压缩全部按钮
  - 空状态提示

#### 2.4.3 压缩提示词模板
- **功能描述**：配置对话压缩时使用的提示词
- **核心功能**：
  - 可用占位符说明：{character_name}, {player_name}, {message_count}
  - 提示词编辑器
  - 保存提示词按钮
  - 重置默认按钮

#### 2.4.4 使用说明
- **功能描述**：记忆管理功能的使用指引
- **核心功能**：
  - 分步使用说明
  - 注意事项提示

### 2.5 运行日志

#### 2.5.1 日志控制
- **功能描述**：日志查看和管理控制
- **核心功能**：
  - 实时/暂停切换
  - 刷新按钮
  - 清空按钮
  - 导出按钮（含下拉选项）

#### 2.5.2 日志筛选
- **功能描述**：日志搜索和筛选
- **核心功能**：
  - 搜索框
  - 级别筛选（全部、INFO、WARN、ERROR）
  - 模块筛选
  - 快速筛选按钮

#### 2.5.3 日志统计
- **功能描述**：日志统计信息
- **核心功能**：
  - 总计数量
  - 错误数量
  - 警告数量

#### 2.5.4 日志列表
- **功能描述**：日志内容展示
- **核心功能**：
  - 日志时间戳
  - 日志级别（颜色区分）
  - 模块标识
  - 日志消息
  - 实时监听标识

│          ├──────────────────────────────────────────────────┤
│  侧边栏  │                                                  │
│  导航    │           主内容区域                             │
│          │                                                  │
│          │                                                  │
├──────────┼──────────────────────────────────────────────────┤

│  区域    │  端口信息（8500-8600范围）                       │

### 3.2 侧边栏导航
- 应用 Logo 和名称
- 作者信息
- 导航菜单（5 个主功能模块）

- 页面标题
- 页面描述
- 功能卡片（分组展示）
- 表单控件
- 实时预览区域（字幕视觉页）

### 3.4 底部区域
- 服务状态显示
- 运行端口信息（显示实际使用的端口）
- 停止服务按钮

### 3.5 系统托盘（桌面应用特性）
- 托盘图标显示
- 右键菜单：显示主窗口、启动/停止服务、退出
- 最小化到托盘
- 托盘图标状态指示（运行中/已停止）

## 5. 核心用户流程

### 5.1 首次使用流程
1. **安装应用** → 运行安装程序
2. **打开应用** → 进入控制面板
3. **配置 API 信息**（URL、Key、模型）
4. **测试连接**
5. **配置语音合成引擎**
6. **启动 Ollama 代理服务**
7. **开始使用**

### 5.2 日常使用流程
1. **启动应用** → 服务自动启动（端口 8500-8600）
2. **选择/配置角色音色**
3. **调整字幕样式**（如需要）
4. **在外部应用中使用 API**
5. **查看日志**（如需要）
6. **关闭应用** → 可选择最小化到托盘或完全退出

### 5.3 安装与卸载流程

#### 安装流程（Windows）
1. 下载安装包（.exe）
2. 运行安装向导
3. 选择安装路径
4. 创建桌面快捷方式（可选）
5. 完成安装并启动

#### 安装流程（macOS）
1. 下载 DMG 文件
2. 拖拽应用到 Applications 文件夹
3. 首次打开授权（系统安全设置）
4. 启动应用

#### 卸载流程
- **Windows**：使用系统卸载程序或安装目录下的卸载程序
- **macOS**：拖拽应用到废纸篓
- **Linux**：使用包管理器卸载

### 5.4 自动更新流程
1. 应用启动时检查更新
2. 发现新版本提示用户
3. 用户确认后下载更新
4. 下载完成后提示重启
5. 重启后应用新版本
