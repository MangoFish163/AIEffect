# AIEffect - 产品设计文档 (PRD)

## 1. 产品概述

### 1.1 产品名称

**AIEffect - AI 交互 Agent 平台**

### 1.2 产品定位

一款基于微服务架构的现代化 AI 交互平台，集 Ollama 代理服务、AI 语音合成（TTS）、语音识别（ASR）、智能字幕展示、对话记忆管理、AI Agents 工作室于一体的综合型 AI 工具，为游戏、视频、桌面应用等提供无缝的 AI 语音集成能力。

### 1.3 核心价值

- 将 Ollama 本地模型转换为 OpenAI 兼容 API
- 提供高质量的 AI 语音合成和字幕展示
- 支持对话历史智能压缩，防止上下文溢出
- 可视化 AI Agents 工作室，支持角色互动编排
- 微服务架构，支持弹性扩展和独立部署

## 2. 功能模块设计

### 2.1 控制面板

#### 2.1.1 Ollama 代理服务

- **功能描述**：在指定端口提供 Ollama 兼容 API 代理服务
- **核心功能**：
  - 服务启停控制（启动/停止）
  - 端口配置（默认 8501）
  - 服务状态实时显示
  - 外部访问地址展示
  - 模型列表获取与展示

#### 2.1.2 API 配置

- **功能描述**：配置 LLM API 接口地址和密钥
- **核心功能**：
  - 服务商预设选择（自定义、OpenAI、Azure 等）
  - 模型 URL 配置
  - API Key 管理（支持显示/隐藏）
  - 模型名称配置
  - 一键复制配置按钮

#### 2.1.3 语音合成

- **功能描述**：语音合成功能开关和基础配置
- **核心功能**：
  - 语音合成总开关
  - 引擎选择（GPT-SoVITS、讯飞语音等）
  - 角色配置（全局默认、男性默认、女性默认）
  - 音量调节
  - 自动播放开关

#### 2.1.4 服务状态监控

- **功能描述**：监控各微服务运行状态
- **核心功能**：
  - API Gateway 状态
  - WebSocket 服务状态
  - 日志服务状态
  - 服务健康检查

### 2.2 字幕视觉

#### 2.2.1 窗口控制

- **功能描述**：控制字幕悬浮窗的显示与行为
- **核心功能**：
  - 显示/隐藏字幕
  - 隐藏控件
  - 清空内容
  - 透明/重置窗口

#### 2.2.2 历史记录

- **功能描述**：查看和管理字幕历史
- **核心功能**：
  - 历史记录列表
  - 记录数量统计
  - 历史记录可在字幕窗口中查看

#### 2.2.3 字幕设置

- **功能描述**：自定义字幕样式
- **核心功能**：
  - 字体颜色自定义
  - 背景颜色选择（含预设如科技极光）
  - 不透明度调节（0-100%）
  - 字体大小调节
  - 打字速度调节（毫秒级）

#### 2.2.4 实时预览

- **功能描述**：实时预览字幕效果
- **核心功能**：
  - 字幕页面预览（Galgame 风格悬浮字幕）
  - 流式窗口预览
  - 测试输入与发送功能

### 2.3 语音配置

#### 2.3.1 功能切换

- **功能描述**：在语音合成（TTS）和语音输入（ASR）之间切换
- **核心功能**：
  - TTS/ASR 标签切换

#### 2.3.2 全局控制

- **功能描述**：语音合成全局设置
- **核心功能**：
  - 启用 TTS 输出开关
  - 自动播放开关
  - 保存音频文件开关
  - 播放模式选择（只播放对话等）

#### 2.3.3 混音控制

- **功能描述**：音频混音设置
- **核心功能**：
  - 主音量调节
  - 背景音乐音量（预留）
  - 音效音量（预留）
  - 停止播放/空闲状态显示

#### 2.3.4 引擎管理

- **功能描述**：管理不同的语音合成引擎
- **核心功能**：
  - 引擎标签页（GPT-SoVITS、讯飞语音等）
  - 连接设置（API 地址、测试按钮）
  - 连接状态显示

#### 2.3.5 角色音色管理

- **功能描述**：配置和管理角色音色
- **核心功能**：
  - 当前角色选择
  - 角色名称配置
  - 参考音频目录配置
  - 情感音频配置
  - GPT 模型路径配置

### 2.4 记忆管理

#### 2.4.1 压缩配置

- **功能描述**：配置对话历史的自动压缩策略
- **核心功能**：
  - 存档目录设置（带浏览按钮）
  - 触发阈值设置（消息数量）
  - 压缩数量设置（保留最近 N 条）
  - 检查频率设置（每 N 条检查一次）
  - 压缩前自动备份开关
  - 自动压缩开关
  - 保存配置按钮

#### 2.4.2 角色状态列表

- **功能描述**：查看和管理各角色的对话历史状态
- **核心功能**：
  - 角色列表展示
  - 刷新状态按钮
  - 压缩全部按钮
  - 空状态提示

#### 2.4.3 压缩提示词模板

- **功能描述**：配置对话压缩时使用的提示词
- **核心功能**：
  - 可用占位符说明：{character_name}, {player_name}, {message_count}
  - 提示词编辑器
  - 保存提示词按钮
  - 重置默认按钮

#### 2.4.4 使用说明

- **功能描述**：记忆管理功能的使用指引
- **核心功能**：
  - 分步使用说明
  - 注意事项提示

### 2.5 Agents 工作室

#### 2.5.1 基础工作室

- **功能描述**：可视化 AI Agents 交互编排
- **核心功能**：
  - 角色列表展示
  - 角色属性配置
  - 基础对话测试
  - 角色状态管理

#### 2.5.2 高级工作室

- **功能描述**：高级 Agents 配置和场景编排
- **核心功能**：
  - 复杂场景配置
  - 角色关系设定
  - 行为规则配置
  - 触发条件设置

#### 2.5.3 PixiJS 可视化工作室

- **功能描述**：基于 PixiJS 的可视化交互界面
- **核心功能**：
  - 可视化角色节点
  - 拖拽式交互编排
  - 实时预览效果
  - 场景动画展示

#### 2.5.4 区域管理

- **功能描述**：管理 Agents 工作室中的交互区域
- **核心功能**：
  - 区域创建与编辑
  - 区域属性配置
  - 区域关联设置

### 2.6 角色管理

- **功能描述**：统一管理所有 AI 角色
- **核心功能**：
  - 角色列表展示
  - 角色创建/编辑/删除
  - 角色属性配置（名称、人设、音色等）
  - 角色导入/导出

### 2.7 运行日志

#### 2.7.1 日志控制

- **功能描述**：日志查看和管理控制
- **核心功能**：
  - 实时/暂停切换
  - 刷新按钮
  - 清空按钮
  - 导出按钮（含下拉选项）

#### 2.7.2 日志筛选

- **功能描述**：日志搜索和筛选
- **核心功能**：
  - 搜索框
  - 级别筛选（全部、INFO、WARN、ERROR）
  - 模块筛选
  - 快速筛选按钮

#### 2.7.3 日志统计

- **功能描述**：日志统计信息
- **核心功能**：
  - 总计数量
  - 错误数量
  - 警告数量

#### 2.7.4 日志列表

- **功能描述**：日志内容展示
- **核心功能**：
  - 日志时间戳
  - 日志级别（颜色区分）
  - 模块标识
  - 日志消息
  - 实时监听标识

## 5. 核心用户流程

### 5.1 首次使用流程

1. **克隆仓库** → 获取项目代码
2. **启动服务** → 运行 `docker-compose up`
3. **打开前端** → 访问 `http://localhost:8500`
4. **配置 API 信息**（URL、Key、模型）
5. **测试连接**
6. **配置语音合成引擎**
7. **启动 Ollama 代理服务**
8. **开始使用**

### 5.2 日常使用流程

1. **启动服务** → 运行 `docker-compose up`
2. **打开前端** → 访问前端页面
3. **选择/配置角色音色**
4. **调整字幕样式**（如需要）
5. **使用 Agents 工作室编排场景**（如需要）
6. **在外部应用中使用 API**
7. **查看日志**（如需要）
8. **关闭服务** → `docker-compose down`

### 5.3 开发环境搭建流程

#### 方式一：Docker Compose（推荐）

```bash
# 1. 克隆仓库
git clone <repository-url>
cd AIEffect

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 文件配置相关参数

# 3. 启动所有服务
docker-compose up -d

# 4. 访问前端
open http://localhost:8500
```

#### 方式二：本地开发

```bash
# 1. 启动日志服务
cd services/log
pip install -r requirements.txt
python start.py

# 2. 启动 API Gateway
cd services/api_gateway
poetry install
poetry run python start.py

# 3. 启动 WebSocket 服务
cd services/websocket
poetry install
poetry run python start.py

# 4. 启动前端
cd frontend
npm install
npm run dev
```

## 6. 技术架构概览

### 6.1 微服务架构

```
┌─────────────────────────────────────────────────────────────┐
│                        前端 (Vite + React)                   │
│  ┌──────────────┐  ┌──────────────┐  ┌─────────────┐       │
│  │  控制面板    │  │  字幕视觉    │  │  语音配置   │       │
│  └──────────────┘  └──────────────┘  └─────────────┘       │
│  ┌──────────────┐  ┌──────────────┐  ┌─────────────┐       │
│  │  记忆管理    │  │  Agents工作室│  │  运行日志   │       │
│  └──────────────┘  └──────────────┘  └─────────────┘       │
└─────────────────────────────────────────────────────────────┘
                              ↕ HTTP / WebSocket
┌─────────────────────────────────────────────────────────────┐
│                      API Gateway (FastAPI)                   │
│         端口: 8501 | 统一入口、路由转发、认证                │
└─────────────────────────────────────────────────────────────┘
                              ↕
        ┌─────────────────────┼─────────────────────┐
        ↓                     ↓                     ↓
┌───────────────┐    ┌───────────────┐    ┌───────────────┐
│  WebSocket    │    │   Log Service │    │  Ollama/      │
│  Service      │    │   (SQLite)    │    │  OpenAI API   │
│  端口: 8502   │    │   端口: 8505  │    │               │
└───────────────┘    └───────────────┘    └───────────────┘
```

### 6.2 服务端口分配（AIEffect 专用区间 8500-8505）

| 端口 | 服务             | 说明                    |
| ---- | ---------------- | ----------------------- |
| 8500 | Frontend         | 前端服务(Vite)          |
| 8501 | API Gateway      | FastAPI 主服务          |
| 8502 | WebSocket Service| WebSocket 服务          |
| 8503 | （保留）         | 字幕服务（未来）        |
| 8504 | （保留）         | TTS 服务（未来）        |
| 8505 | Log Service      | 日志服务                |
